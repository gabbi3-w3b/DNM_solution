How to Use

    Steps
  1. Identify the equation solution type by clicking (Click here to find out).
  2. Then click the option
    • Formula Method
    • Factorization Method
    • Completing the Square Method
    according to your equation form or how you wish to solve.
  3. In the options, there are normally three options (a, b, c, and an optional variable).
  4. Set the values (numerical).
  5. Click the Solve button.

Note

  • If the variable option is left empty and the solve button is clicked, a default value will be set for it.
  • When it is necessary to leave the input box empty, insert zero ('0' for blank) and one ('1' for a variable to be multiplied by 1).
